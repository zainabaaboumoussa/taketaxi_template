Progressive Web App

Pourquoi cette technologie mobile dernière génération est une révolution pour les VTC ?

Installation immédiate

Pas de temps de chargement , l'ajout à l'écran d’accueil se fait en un clic, idéal pour partager avec vos clients .

Hyper légère

Une progressive Web App est très légère, elle utilise 200 fois moins de mémoire qu'une application native

Accessible comme un app

L'icone de l'app apparaît sur l'écran d’accueil comme une application native.

Fonctionne hors connexion

Les informations importantes sont toujours disponible. Vos clients ont accès à vos forfait et vos coordonnées quelque soit la qualité de leur réseau.

Notification Push

Les progressive Web App permettent de recevoir des notifications push idéal pour confirmer les points des cartes fidélité.

Géolocalisation

L'utilisateur est géolocalisé, ce qui nous permet de trouver des chauffeurs autour de lui.
